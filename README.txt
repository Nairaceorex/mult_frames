0) Ставим интерпретатор.
1) Открываем проект в PyCharm. Проект - это папка pz_2(в pz_2 должен быть main.py)
1.1) Даём путь до интерпретатора (нажимаем справа внизу на No interpreter => Interpreter Settings => жмём add interpreter => Add local Interpreter => System Interpreter => Выбираем один из путей => Нажимаем Ok.
Если нет надписи No interpreter, то значит необходимости делать этот шаг нет.
Если вообще пусто - ставим python

2) Нажимаем alt+F12 (открывается консоль) и пишем:
pip install -r requirements.txt
3) После всех установок пишем команду:
python main.py
Тыкаем 
4) Задание выполнено
